## 🚀 Featured Project

### [Birdple Test Automation Framework](https://github.com/ipenha9/birdple-test-automation)
End-to-end automation framework for a web platform built with Python, Selenium WebDriver, and Pytest using Page Object Model architecture.

- ✅ Multi-browser support (Chrome, Firefox)
- ✅ POM with separated locators
- ✅ Reusable fixtures via conftest.py
- ✅ Multi-environment config (test / prod)
- 🚧 E2E and UI suites in progress
